# Tank War

Tank War is my first game, created in two weeks with a colleague. 
Tank War is the first game developed at the school. As part of a two-week project in pairs, we had to develop the game concept, source assets, and design the gameplay. All of this was built using an engine we developed during class to prepare us for using Unreal Engine. 

## What Tank War Offers


- Customize your tank by choosing the tracks, hull, and turret.
- Semi-procedural generation within a zone
- A fierce battle against your opponent using shells
- A tank that can become overpowered 
- A collection of funny and entertaining bugs.


Tank War was the very first project created at the school, and it was the one that confirmed to me that video games were indeed what I wanted to make in the future—to create entertainment and see people having fun with the games we can build!
